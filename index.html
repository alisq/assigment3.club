In the wake of the COVID-19, OCADU University has had to shift to online classes, leaving students in the lurch. Amidst them, Typography 3 students will not have the opportunity to develop their third assignment for the course. To compensate and to provide a learning resource for anyone at this time, instructors Chris Lange and Ali S. Qadeer will use this site as both an assignment and tool.

Let’s use this time to explore the possibility of HTML & CSS as a typographic medium. Through a series of exercises, we’ll learn the fundamentals of both syntaxes, practice and get comfortable with a way of making that is wholly different from what we’re used to (especially pertaining to typography), and collaborate a little here and there.

Through Zoom lectures, tutorials, and additional online resources we will work through basic concepts and implementation of the HTML & CSS to build unique outputs from week to week.

Part 1: CSS Typeface

To begin to get you comfortable with the use of code to create form, we will start with a drawing exercise. Create a typeface with only HTML and CSS. It should be a minimum of 26 characters. All characters should live on a single HTML page. So we can re-deploy these forms in future work, ensure that all the DIVs you create use position:absolute within an assigned character div.

A typeface is a consistent formal system so think about things like scale, proportion, stroke, serif, angle, contrast, etc.

CSS Typeface Template If you want, you can use this file to begin the first part of the assignment

Examples: Rave in berlin (CSS mtypeface)

Part 2: HTML Typographic Poster

Alongside traditional methods of web typesetting, deploy your CSS typeface on a single HTML page to create a web typographic poster. You can make anything you want: an event schedule, a large statement, a printed form. You decide as long as it contains both your build typeface as well as traditional layers of web typography.
